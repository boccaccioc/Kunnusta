Data

The API that I am using is called Yahoo Finance Low Latency. This API returns data about the stock in the query, such as price, ticker, trading hours, and price points throughout different times of the most recent day's trading. From this data, I return from my server, upon query of a equity name, the ticker of the stock, the price, the date that the data is from, and two arrays (times and the prices that they map to). Unfortunately, as of very recently, I cannot access this JSON data due to an excess of API calls for the day, but the link to the API is here (https://rapidapi.com/finance.yahoo.api/api/yahoo-finance-low-latency?endpoint=apiendpoint_0361945e-e27c-4b15-a61e-1deddb4ef50a) and allows for "Testing Endpoints" without signup, which will show what JSON is returned.
