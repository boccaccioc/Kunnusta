The Data

The API used to retrieve the initial financial data is "Yahoo Finance Low Latency". Calls are made to this API per stock, with the stocks being those saved by the user, or if through a query using the search bar. User Data is stored in firebase, with the data of each user consisting of their saved stocks (These stocks are indexed using numbers initially so that the stocks can be interacted with as an array) and the admin status of the user (true of false)


How it Works

This app retieves data from the api based on the user that is currently signed in (User data stored in firebase). Users sign in using Google's OAuth, and then have the ability to add, delete, and change the order of stocks on the home screen. These actions modify a local array, which is then pushed to heroku, which then updates the firebase data of that user.
